ME

I like to do programming, designing websites, and learning. And I love to talk about Jesus, and the Bible. So I decided to mash these things together to make this. I wanted to make a github project that can provide scripture using javascript.
I like the style that I went for is an old computer type style. I chose old bios type fonts (along with monospace, courier new, and Courier New.)


Project: https://github.com/bib-notes/bib-notes.github.io

Website

This github wepage is designed to appear like an old computer interface. I have a fascination with the look and feel of old computer Graphical User Interfaces. And I couldn't decide on what colors I should use, so I made 3 theme colors:

  • Red
  • Green
  • Blue
And I created 3 buttons on the footer bar so the user can switch between the different colors. I was planning on making a typewriter effect, but it could take a while. Credits
